# 📚 BookNest

BookNest is a Full Stack Book Tracking Web App where users can add, edit, delete, search, and sort books they have read.

## 🚀 Features
- Add New Books
- Edit Book Details
- Delete Books
- Search Books
- Sort by Latest / Rating / Title
- Book Covers using Open Library API

## 🛠 Tech Stack
- Node.js
- Express.js
- PostgreSQL
- EJS
- HTML/CSS/JavaScript

## ▶ Run Project

npm install
nodemon index.js
